# 中国 A 股上市公司全景数据深度分析 (EDA)

## 📖 项目简介
本项目是一个完整的端到端（End-to-End）金融数据工程与分析实战项目。项目旨在通过编写自动化 Python 脚本，从互联网动态抓取中国 A 股市场（沪、深、北三大交易所）**全部在市**公司的全量基础名录，并通过内置的特征降维算法清洗数据，最后基于清洗后的高质量数据展开多维度的探索性数据分析（EDA）与可视化。

本项目无需依赖任何本地离线的商业数据库文件，完全通过代码实现了从“网页杂乱数据”到“机构级标准数据”的蜕变。

## 🎯 核心分析维度
数据清洗完毕后，项目从以下四大视角透视中国资本市场：
1. **中国资本市场改革历程（事件研究）**：将各年度 IPO 数量趋势与重大政策节点（如股权分置改革、注册制试点等）相叠加，让数据讲述政策背后的故事。
2. **各板块年度新增上市结构（扩容主力）**：利用交叉表与堆叠柱状图，透视每年新增上市公司中主板、创业板、科创板及北交所的占比变迁。
3. **各板块“上市年龄结构”（老龄化透视）**：计算各公司的绝对上市年龄，利用箱线图和密度图对比不同板块的历史厚度与新老交替差异。
4. **全市场及各交易所行业分布画像（基石透视）**：展现 A 股整体的宏观行业聚集度，并对比三大交易所在行业定位与吸引力上的差异化格局。
